Instructions to unbundle and configure The GlassFish Eclipse plugin

GlasssFish Eclipse Plugin installation requires JDK 1.4 or 5, and an existing Eclipse and WTP 1.0 installation.
  1. For Milestones 0.2.1, 0.3, 0.4 and later, download the zip file and just unzip it in the Eclipse directory. (This dir should have a sub directory called "plugins")
  2. For the Milestones 0.2 and 0.1: Download the latest plugin installable JAR file and save it (Do not expand it, it is an installer)
  3. Run: 
    java -jar glassfish-eclipse-plugin_installer_0_2.jar
  4. Accept the CDDL licence
  5. In the File Chooser, navigate to the installation directory of Eclipse 3.x that also contains WTP 1.x (i.e C:\EclipseInstallDir\eclipse) (this directory usually contains a sub-directory called plugins)
  6. Click the Select button: the plugin is installed.

Read the GlassFish Eclipse Plugin Quick Start Quide for basic steps to use this new Eclipse plugin.

GlassFish Tools Bundle for Eclipse v1.0 (May 15, 2009)
Description: This is the most stable recent distribution of the GlassFish Tools Bundle for Eclipse. It contains the Eclipse 3.4.2 Java EE IDE, GlassFish v2.1 and GlassFish v3 Prelude pre-configured, and optionally, a JDK 1.6. (It is still possible to get the GlassFish Eclipse Plugin independently).
License: The GlassFish Tools Bundle for Eclipse code is available under the Dual License Common Development and Distribution License (CDDL) v1.0. and GPL v2 with ClassPath Exception. For the Eclipse IDE itself, refer to the Eclipse EPL license.

Prerequisities: If you are selecting an installer without the JDK, you will need a Sun JDK to be pre-installed: version 1.5.0_14 or later for JDK 5, or version 1.6.0_03 or later for JDK 6. The Mac installer does not bundle a JDK 1.6.

Installation instructions:

  • Download the installer for your platform, and run the installer. Follow the instructions on screen.
  • Note: pay attention to the port numbers required for GlassFish: make sure that the port numbers are not used by other applications.
  • When the Registration Screen appears, please register. Registration benefits include improved support information in the GlassFish administration console, access to whitepapers such as high availability and performance tuning, access to past and future webinars, and more.

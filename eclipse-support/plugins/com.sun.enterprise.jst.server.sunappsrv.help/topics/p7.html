Appendix A

Using Popup Menu

The Popup Menu available in the GlassFish Tools Bundle for Eclipse provides a number of usage options, including customized options for GlassFish Server. To access the Popup Menu from Eclipse IDE, select Servers view, and right click mouse.

The following options are available in the Popup Menu:

New

Allows the user to create a new GlassFish Server instance.

Open (F3)

Allows the user to access Server Overview Screen. The Server Overview Screen allows the user to view and modify some GlassFish Server properties:

  • Specify hostname and other common properties

  • Modify settings for Publishing

  • Specify the time limit to complete Server operations

  • Edit some of the Server runtime properties

Show In (Shift+Alt+W)

Allows the user to direct the GlassFish Server log messages to Console or Debug windows. This option is also accessible through the (Shift+Alt+W) action.

Copy (Ctrl+C)

Allows the user to copy the GlassFish Server Instance. This option is also accessible through the (Ctrl+C) action.

Paste (Ctrl+V)

Allows the user to paste the copied GlassFish Server Instance. This option is also accessible through the (Ctrl+V) action.

Delete (Delete)

Allows the user to delete the selected GlassFish Server Instance from the configuration. This option is also accessible through Delete button from the keyboard.

Rename (F2)

Allows the user to rename the GlassFish Server Instance. This option is also accessible through the F2 function key.

Debug (Ctrl+Alt+D)

Allows the user to start the GlassFish Server Instance in Debug mode. This option is also accessible through the (Ctrl+Alt+D) action.

Start (Ctrl+Alt+R)

Allows the user to start the GlassFish Server Instance. This option is also accessible through the (Ctrl+Alt+R) action.

Profile

This option is not implemented in the GlassFish Tools Bundle for Eclipse.

Stop (Ctrl+Alt+S)

Allows the user to stop the GlassFish Server Instance. This option is also accessible through the (Ctrl+Alt+S) action.

Publish (Ctrl+Alt+P)

Allows the user to deploy the built applications to the GlassFish Server. This option is also accessible through the (Ctrl+Alt+P) action.

Clean

Allows user to discard all published state from the GlassFish Server and republish from scratch.

Add and Remove Projects

Allows the user to add or remove Projects that are configured to run on the GlassFish Server.

Monitoring

Allows the user to monitor GlassFish Server ports for access. This is a new feature available in GlassFish Server v3 Prelude.

GlassFish Enterprise Server

Allows the user to access to the following additional sub menu options:

  • GlassFish Update Center

  • Register the GlassFish Enterprise Server

  • Get Support for GlassFish Enterprise Server

  • Aquarium: GlassFish Community News

  • View Admin Console

  • View Log File

Properties (Alt+Enter)

Allows the user to view the GlassFish Server properties and modify some of them. This option is also accessible through the (Alt+Enter) action.

Chapter 2

Installing GlassFish Tools Bundle for Eclipse

This section of the document describes the process of installing GlassFish Tools Bundle for Eclipse software includes Eclipse IDE, GlassFish Server v2.1 and v3 Prelude, and the plug-in that integrates these products on different Operating Systems.

The following tasks describe the process of installing the product on different platforms.

  • To Install on Windows Platform

  • To Install on Mac OS Platform

  • To Install on Linux Platform

  • To Uninstall

  • Troubleshooting

To Install on Windows Platform

  1. Download GlassFish Tools Bundle for Eclipse Windows package from here: http://download.java.net/glassfish/eclipse
  2. Double click the downloaded file, to start the installer wizard.

    Welcome Message is displayed.

  3. Click on Next to continue.
  4. In the License Agreement screen, click on I agree to continue.
  5. In the next screen, accept the default location or select a different location to install software.

    For example: C:\GlassFish-Tools-Bundle-For Eclipse-<version>\

  6. Click on Install to continue.

    Wait for the installation to complete.

    When installation is completed, Eclipse IDE 3.4 and GlassFish Servers v2.1 and v3 Prelude along with the plug-ins that integrates these products are installed on the system.

  7. Click on Close to quit the Installer wizard.
  8. To start Eclipse IDE, navigate to Start Menu. select Programs->GlassFish Tools Bundle<version> for Eclipse->GlassFish Tools Bundle for Eclipse.
  9. Accept default workspace or select a different workspace, when prompted.
  10. Register the GlassFish Server from the Registration Screen.

    Eclipse IDE is launched.

    Tip - By registering your GlassFish Installation, you have the advantage of receiving updates, fixes and support from GlassFish community.

  11. Close the Welcome page to enter the default Workbench, in Java EE Perspective.

To Install on Mac OS Platform

  1. Download GlassFish Tools Bundle for Eclipse Mac OS package from here:http://download.java.net/glassfish/eclipse
  2. Double click the downloaded file to create the pkg file
  3. Click the pkg file to start the installer wizard.

    Welcome Message is displayed.

  4. Click on Continue to continue.

    License Agreement is displayed.

  5. Click on I agree to continue.
  6. Accept the default location or select another location to install software.

    For example: HD Disk

  7. Click on Install to continue.

    Wait for the installation to complete.

    When installation is complete, Eclipse IDE 3.4 and GlassFish Servers v2.1 and v3 Prelude along with the plug-ins that integrates these products are installed on the system.

  8. To start Eclipse IDE, select the GlassFish Tools Bundle for Eclipse from Start menu.
  9. Select a workspace or accept default workspace, when prompted.
  10. Register the GlassFish Server from the Registration Screen.

    Eclipse IDE is launched.

    Tip - By registering your GlassFish Installation, you have the advantage of receiving updates, fixes and support from GlassFish community.

  11. Close the Welcome page to enter the default Workbench, in Java EE Perspective.

To Install on Linux Platform

  1. Download GlassFish Tools Bundle for Eclipse Linux package from here: http://download.java.net/glassfish/eclipse
  2. Move the file to a suitable directory, gunzip and extract the tar file.

    When the file is extracted, Eclipse IDE 3.4 and GlassFish Servers v2.1 and v3 Prelude along with the plug-ins that integrates these products are installed on the system.

  3. Navigate to the install location, <glassfish-tools-bundle-for-eclipse-xxxx> directory.

    The directory contains Eclipse IDE, GlassFish Server v2.1 and v3 Prelude installations.

  4. Start Eclipse IDE, by using the following command:

    ./eclipse

  5. Register the GlassFish Server from the Registration Screen.

    Tip - By registering your GlassFish Installation, you have the advantage of receiving updates, fixes and support from GlassFish community.

  6. Install wizard configures GlassFish Server v2.1 and v3 Prelude Servers.
  7. Select a workspace or accept default workspace, when prompted.

    Eclipse IDE is launched.

  8. Close the Welcome page to enter the default Workbench, in Java EE Perspective.

To Uninstall

To uninstall your GlassFish Tools Bundle for Eclipse installation, use the following procedure:

  1. For Windows installations, select Start->GlassFish Tools Bundle For Eclipse <version>->GlassFish Tools bundle Uninstall.
  2. For Mac OS installations, go to <Disk>/Applications directory, remove the <installation>directory.
  3. For Linux installations, remove the <installation> directory.

Troubleshooting

If you have any issues in installing the bundle, use the following steps to clean up the failed install.

  1. Remove the <installation> directory.
  2. Remove the hidden directories and files from you home directory.

    For example: .glassfish-tools-bundle-for-eclipse-<version> on Linux.

  3. Remove the workspace if configured.
  4. Reinstall the GlassFish Tools Bundle for Eclipse.

Instructions to unbundle and configure The GlassFish Eclipse plugin

GlasssFish Eclipse Plugin installation requires JDK 1.4 or 5, and an existing Eclipse and WTP 1.0 installation.
  1. Download the latest plugin installable JAR file and save it (Do not expand it, it is an installer)
  2. Run: 
    java -jar glassfish-eclipse-plugin_installer_0_1.jar
  3. Accept the CDDL licence
  4. In the File Chooser, navigate to the installation directory of Eclipse 3.1 that also contains WTP 1.0 (i.e C:\EclipseInstallDir\eclipse) (this directory usually contains a sub-directory called plugins)
  5. Click the Select button: the plugin is installed.

See the GlassFish Quick Start Quide for basic steps to use this new Eclipse plugin.

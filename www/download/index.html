Milestone binary builds for the Eclipse IDE

Eclipse 3.3 and newer

The majority of the GlassFish Plugin for Eclipse code is available under the Common Development and Distribution License (CDDL) v1.0 and the GNU General Public License (GPL) v2. The following page contains details about third party code included in the Plugin.
Here are the release notes to help you choose which version to use.
  • The best way to install the glassfish plugin for eclipse is from within Eclipse itself. Open the New Server wizard, then click the Download additional server adapters link. See this page for more details.
  • If you already have a version of the plugin installed, you can update to a newer version in one of two ways:
    1. Use the New Server wizard again, and choose the Download additional Server adapters again - this will download the latest version
    2. Use the Help->Software Updates menu item. The update site for the glassfish plugin is automatically added to the list of managed sites once you install the plugin, but it is not automatically checked as active. See Issue 45 for more details. Note that the Software Updates wizard in Eclipse 3.4 allows you to set an option which shows you older versions of updates (Turn off the "Show only the latest versions of available software" checkbox).
  • If you need to install an older version of the plugin, and the Software Updates wizard doesn't work for you, you can choose to install it manually.
    1. Choose a version from this table
    2. Download both the plugin and feature jar files
    3. Quit Eclipse, then copy the unzipped contents of the first into the plugins directory and the second into the features directory of your Eclipse installation.
    4. Restart Eclipse and you should be ready to go.

Eclipse 3.2 and older

  • Milestone 0.4- glassfish-eclipse-plugin_installer_0_4.zip January 2007. (Work with Eclipse 3.2 WTP 1.5 current versions. Works on Linux and Mac OSX as well)
  • Milestone 0.3- glassfish-eclipse-plugin_installer_0_3.zip June 3, 2006. (Work with Eclipse 3.2 WTP 1.5 current versions. Works on Linux and Mac OSX as well)
  • Milestone 0.2.1- glassfish-eclipse-plugin_installer_0_21.zip April 6, 2006. (Work with WTP 1.0 and 1.0.1 latest. Works on Linux as well)
  • Milestone 0.2- glassfish-eclipse-plugin_installer_0_2.jar February 14, 2006. (Work with WTP 1.0 and 1.0.1 latest. Works on Linux as well)
  • Milestone 0.1- glassfish-eclipse-plugin_installer_0_1.jar January 3, 2006.

    • Instructions to unbundle and configure The GlassFish Eclipse plugin

    GlasssFish Eclipse Plugin installation requires JDK 1.4 or 5, and an existing Eclipse and WTP 1.0 installation.
    1. For Milestones 0.2.1, 0.3, 0.4 and later, download the zip file and just unzip it in the Eclipse directory. (This dir should have a sub directory called "plugins")
    2. For the Milestones 0.2 and 0.1: Download the latest plugin installable JAR file and save it (Do not expand it, it is an installer)
    3. Run: 
      java -jar glassfish-eclipse-plugin_installer_0_2.jar
    4. Accept the CDDL licence
    5. In the File Chooser, navigate to the installation directory of Eclipse 3.x that also contains WTP 1.x (i.e C:\EclipseInstallDir\eclipse) (this directory usually contains a sub-directory called plugins)
    6. Click the Select button: the plugin is installed.

Read the GlassFish Eclipse Plugin Quick Start Quide for basic steps to use this new Eclipse plugin.

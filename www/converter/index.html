Eclipse Import Plug-in for Netbeans Glassfish Projects

Download the Eclipse Plug-in
or use the update site: coming soon...

Created by Michael Tidd

Prerequisite:

Download the GlassFish Tools Bundle for Eclipse for the quickest way to start with these components.


Getting Started:

Import the project using either File-Import or right-click Import-Import

Select Glassfish-"Import Glassfish Project from Netbeans"


Browse for the Glassfish project you would like to import


Select the project and click Choose


The project can be seen in the Project Explorer
